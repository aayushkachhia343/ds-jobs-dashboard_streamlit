# Data Science Jobs — Mini Salary Dashboard (Streamlit)

This is a **small hands-on project** I built using **Streamlit** to practice creating a modern, interactive dashboard for exploring data science job salaries.  
**Thank you for taking a look!**

---

## What this dashboard shows
Using the **Data Science Jobs Salaries** dataset, the app lets you explore:
- Salary distribution (USD)
- Salary by experience level
- Top job titles by count
- Remote ratio vs salary
- Filtered dataset view + download

---

## Features
- ✅ Clean, modern UI (no sidebar)
- ✅ Top controls (Year, Experience, Remote, Top-N titles)
- ✅ KPI cards (Rows, Median Salary, Average Salary, Remote share)
- ✅ Interactive Plotly charts
- ✅ Download filtered results as CSV
